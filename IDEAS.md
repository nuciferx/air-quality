# Ideas — แก้ปัญหาฝุ่นเยอะ

## ปัญหาหลัก
ยังไม่รู้ว่าฝุ่นมาจากไหน เมื่อไหร่ และแต่ละห้องต่างกันยังไง

---

## 1. Alert — แจ้งเตือนเมื่อฝุ่นเกิน threshold
**เป้าหมาย:** รู้ทันทีที่อากาศแย่ ไม่ต้องเปิด app ดูเอง

**วิธีทำ:**
- Cloudflare Worker cron ทุก 5 นาที เช็ค PM2.5 แต่ละห้อง
- ถ้าเกิน threshold → ส่ง Telegram แจ้งเตือน
- ถ้ากลับมาปกติ → แจ้งอีกครั้ง (all clear)

**Threshold แนะนำ:**
| ระดับ | PM2.5 | สี |
|-------|-------|-----|
| ดี | < 15 | 🟢 |
| ปานกลาง | 15–35 | 🟡 |
| แย่ | 35–75 | 🟠 |
| อันตราย | > 75 | 🔴 |

**ตัวอย่าง Telegram:**
```
🟠 ฝุ่นเยอะ — โถงชั้นล่าง
PM2.5 = 48 µg/m³ (เกิน 35)
อุณหภูมิ 29°C | ความชื้น 65%
```

## 2. Morning Report — สรุปอากาศทุกเช้า
**เป้าหมาย:** รู้ภาพรวมคืนที่ผ่านมาตอนตื่นนอน

**วิธีทำ:**
- GitHub Action cron `0 1 * * *` (08:00 น. ไทย)
- ดึงข้อมูลจาก `/api/history/stats?hours=24`
- ส่ง Telegram สรุปแต่ละห้อง + Qwen AI วิเคราะห์

**ตัวอย่างข้อความ:**
```
🌅 รายงานอากาศ — 13 เม.ย. 2026

ห้องทำงาน      PM2.5 เฉลี่ย 12 | สูงสุด 28
ห้องนอนชั้น 2  PM2.5 เฉลี่ย  8 | สูงสุด 15
โถงชั้นล่าง    PM2.5 เฉลี่ย 22 | สูงสุด 51
ห้องแมวชั้น 2  PM2.5 เฉลี่ย 30 | สูงสุด 68

⚠️ Filter ใกล้หมด: ห้องแมว 18%

🤖 อากาศโถงชั้นล่างแย่กว่าวานเกือบ 2 เท่า
ควรเช็คว่ามีกิจกรรมพิเศษหรือหน้าต่างเปิดอยู่
```

## 3. Outdoor vs Indoor — เทียบฝุ่นนอก/ในบ้าน
**เป้าหมาย:** รู้ว่าฝุ่นมาจากข้างนอกหรือเกิดในบ้าน

**วิธีทำ:**
- ดึงข้อมูล PM2.5 ภายนอกจาก AQICN API (ฟรี)
- เทียบกับค่าในบ้านแบบ realtime
- ถ้าในบ้าน > นอกบ้าน → แหล่งฝุ่นอยู่ในบ้าน (ทำอาหาร? สัตว์เลี้ยง?)
- ถ้านอกบ้าน > ในบ้าน → ฝุ่นรั่วเข้ามา (ปิดหน้าต่าง)

**Station ใกล้บ้าน:** ดึงจาก `api.waqi.info`

## 4. Auto Control — เปิด/ปิดอัตโนมัติตามระดับฝุ่น
**เป้าหมาย:** ประหยัดไฟ + อากาศดีตลอด โดยไม่ต้องสั่งเอง

**Logic:**
```
PM2.5 > 35  → เปิดเครื่อง + ตั้ง mode Favorite (กำลังแรง)
PM2.5 15-35 → เปิดเครื่อง + mode Auto
PM2.5 < 15  → ปิดเครื่อง (ประหยัดไฟ)
```

**วิธีทำ:**
- Worker cron ทุก 5 นาที อ่านค่า PM2.5
- เปรียบเทียบกับ threshold
- ส่ง `/api/control` สั่งเครื่องอัตโนมัติ
